Emergency Site

In this theme, I worked with the development of user interfaces using HTML, CSS, and JavaScript. The focus was on combining visual design with functional frontend solutions. I explored interactive elements and basic accessibility considerations.

The final solution is a concept website. "Emergency in Ikea" The interface consist of multiple features such as "forms, Modals, pop-ups and Javascript implementations"

The project began with an ideation phase where concepts and visual directions were explored. Ideas were developed with help from exercises such as, "Crazy Eight, Brainstorm and peer to peer" and was in the end collected which gave the building blocks for a styletile defining colors, typography, and other visual elements.

Custom illustrations and visual elements were developed to support the visual identity of the site. These elements were integrated as part of the user interface to enhance engagement. The visuals were adapted to match the overall style.

A lot of the copy and images were developed using Ai. - ChatGPT for copy and Adobe Firefly for images. Through trial and error i tried to changes the prompt to get the best match of the overall style of the website.

This image provides an overview of different interface components and their visual states. It also illustrates the difference between Light and Dark mode, where the colors have been desaturated to create a darker visual appearance.

The interface consists of a form developed using HTML, CSS, and JavaScript, including modals and interactive JavaScript functionality.
