Emne Site

This project focuses on UX/UI and how user research and testing informed the design process. I worked with moodboards, wireframes, styletiles, and prototypes to shape both structure and visual direction. The final solution was developed using HTML and CSS, based on a digital prototype. Throughout the process, usability, clarity, and the overall user experience were key priorities.

The final website was developed based on UX principles and digital prototypes. I translated research and design decisions into a functional site using HTML and CSS. The focus was on usability, structure, and a clear user journey.

User research was used to gain insights into user needs and behavior. These insights formed the foundation for design decisions throughout the process. Research helped ensure that the solution was based on real user understanding.

Moodboards and visual research were used to explore style, tone, and inspiration. This process helped define the visual direction of the project before moving into detailed design. Visual decisions were aligned with the theme and target audience.

Wireframes and digital prototypes were developed in Figma to structure the user interface. The design was tested using methods such as think-aloud tests and 5-second tests. Feedback from testing was used to iterate and improve the user experience.
